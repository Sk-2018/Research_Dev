
# Payload Diff Viewer v1.0.3
- Adds **Tools → Select Columns…** dialog (always available)
- Adds **Tools → Choose Row…** to pick which row (by Config Key/Name) to compare
- Smarter auto-detection: scans top rows for JSON-looking payload columns

## Install
pip install "deepdiff>=6.7.1" "pandas>=2.0.0" "openpyxl>=3.1.2" "matplotlib>=3.8.0" "orjson>=3.9.15"

## Run
python run_app.py
